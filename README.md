# Browser-homepage

一个现代化的浏览器启动页，提供简洁美观的界面和实用功能。

## 功能特性

- ⏰ **实时时钟和日期显示** - 自动更新的时间和日期
- 🔍 **智能搜索** - 支持多个搜索引擎（Google、百度、Bing、DuckDuckGo）
- 🔗 **快捷链接** - 预设常用网站，支持自定义添加
- 💾 **本地存储** - 自定义链接保存在浏览器本地
- 🎨 **渐变背景** - 美观的渐变色背景
- 📱 **响应式设计** - 适配各种屏幕尺寸
- ⌨️ **键盘快捷键** - 按 `/` 快速聚焦搜索框

## 使用方法

1. 下载或克隆本仓库
2. 在浏览器中打开 `index.html` 文件
3. 将此页面设置为浏览器的启动页或主页

### 设置为浏览器主页

#### Chrome/Edge
1. 打开设置 → 启动时 → 打开特定网页或一组网页
2. 添加本地文件路径：`file:///path/to/index.html`

#### Firefox
1. 打开设置 → 主页 → 自定义网址
2. 输入本地文件路径：`file:///path/to/index.html`

## 自定义

- **添加快捷链接**：点击"添加链接"按钮，输入标题和URL
- **搜索引擎**：从搜索框左侧的下拉菜单选择
- **背景设置**：点击右下角的设置按钮（功能开发中）

## 技术栈

- HTML5
- CSS3 (使用 Flexbox 和 Grid)
- 原生 JavaScript (无依赖)
- LocalStorage API
